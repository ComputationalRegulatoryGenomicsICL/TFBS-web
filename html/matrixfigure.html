The types of matrix patterns and part of the TFBS functionality associated with them: (a) Position frequency matrix is represented by TFBS::Matrix::PFM, and is usually derived from a set of fixed-size transcription factor binding sites (b) by counting the number of occurrences of each nucleotide at each position. In TFBS this is performed with TFBS::PatternGen::SimplePFM, a class used to extract the pattern from a set of sequences. (c) An information content matrix (TFBS::Matrix::ICM object) can be obtained by calling to_ICM() on a TFBS::Matrix::PFM object, and is used e.g. for calculating the total information content of a matrix, which is a measure of specificity of the pattern. It can also be directly visualized as a sequence logo (d), which is drawn by calling the draw_logo() method; (e) A position weight matrix can also be derived from a frequency matrix; it is the matrix preferentially used for the analysis of nucleotide sequences (search_seq() method) or pairwise alignments with conservation filtering (search_aln()). The result of scanning a sequence by search_seq() is a TFBS::SiteSet object (f), an aggregate of TFBS::Site objects representing the actual binding sites. The set can be analyzed as a whole, or its component sites can be manipulated individually. It can also be output e.g. as a list of sequence features in GFF format (g).
